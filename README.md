# python3
python3系でマルコフ連鎖による文書生成（50文字）
<p>環境　Requirements</p>
<ul>
<li>Linux Mint</li>
<li>python3</li>
<li>mecab-python3</li>
</ul>
<p>動作方法</p>
<ul>
<li>text.txtファイルになるべく長めの文章を書く</li>
<li>text.txtファイルをmarcov.pyと同じディレクトリに配置</li>
<li>python marcov.pyでターミナルから実行</li>
</ul>
